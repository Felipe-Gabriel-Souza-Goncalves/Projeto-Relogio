## ✏️ Descrição
Pequeno projeto para uso de relógio, timer e cronômetro

## ⚙️ Funcionalidades
- Relógio
  - Formato 24h e 12h
- Timer  
  - Adicionar períodos de tempo (5seg, 30seg, 60seg)
  - Pausar
- Cronômetro
  - Cronometrar
  - Pausar
  - Marcar tempo
